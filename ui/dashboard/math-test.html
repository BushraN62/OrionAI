This page demonstrates how mathematical equations will appear in the Orion Dashboard. All equations are rendered using KaTeX, the same library used in the main application.

How to use math in Orion:
  • Inline math: Wrap in single dollar signs: $x^2 + y^2 = z^2$
  • Display math: Wrap in double dollar signs: $$\int_0^1 x^2 dx$$
  • Greek letters: Use backslash: $\alpha$, $\beta$, $\gamma$
  • Fractions: Use \frac{numerator}{denominator}
  • Subscripts/Superscripts: Use _ and ^
